# The Robot Byte
### {3rd Position in COFAS International's Game Dev Competition "Softek"}
This is a 2D pixel-style RPG game. The story revolves around a boy who moves into a new house and teleports to a new world. This world has a few challenges for him and making him succeed is your job.



## Controls:
Movement: WASD
Attack (with choosing directions) : Arrows
Using Health Potion: H
Using Invincibility Potion: I
Pausing: Esc
